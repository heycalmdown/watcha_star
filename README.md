# watcha_star

더 왓차 별점 가져오머


## 기본 동작

- 유저 별 `watcha` 정보를 긁어와서 `cards.json`으로 백업을 해놓는다
- 영화 타이틀과 별점만 뽑아서 출력한다


## Options

- user: `watcha` 유저 키 (`HpXEcsbL0ZkX` 이렇게 생겼다. 알아서 잘 구하시길)
- output
  - console
  - csv
  - markdown
- update: `cards.json`이 있어도 강제로 긁어옴
